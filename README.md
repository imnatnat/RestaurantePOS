# RestaurantePOS

Proyecto Integrador de Aprendizaje del Eq. 4 — Clase de Bases de Datos
Propósito: demostrar el uso de bases de datos en sistemas y aplicaciones de la vida real.
Este es un sistema POS para un restaurante construido con HTML, CSS y JavaScript vanilla, usando Firebase Firestore como base de datos en tiempo real.
Cuenta con tres roles de usuario: Hostess (mapa de mesas y reservaciones), Server (toma de órdenes y envío a cocina) y Manager (dashboard, staff y reportes). Todo el sistema vive en un solo archivo index.html.
